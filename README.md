# Marmot (土拨鼠) — Lightweight Alert Framework for Python

A developer-friendly, zero-dependency alert framework focused on **simplicity** and **practicality**.

## Installation

```bash
pip install marmot-alert
```

## Quick Start

```python
import marmot

marmot.configure("alerts.db")
print(marmot.__version__)
```
